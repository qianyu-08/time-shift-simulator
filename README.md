```md
# Time Shift Simulator

ブラウザ上で“タイムトラベルしてる気分”だけを味わうためのUIです。  
実際にはどこにも移動しません。残念ながら物理法則はまだこちら側に味方しています。

## 概要
このプロジェクトは、日時を選択すると
- 過去っぽい雰囲気
- 未来っぽい雰囲気
- それっぽいログ演出

を出して「時空移動した気分」にさせるだけのシミュレーターです。

## 特徴
- タイムジャンプ演出（ただのアニメーション）
- 年代ごとの雰囲気テキスト生成
- ランダム時空ジャンプ（無責任）
- 安定度メーター（それっぽいだけ）
- ログ表示（それっぽい記録）

## 使い方
1. `index.html` を開く  
2. 日時を選ぶ  
3. 「タイムジャンプ開始」を押す  
4. 脳内で時空が歪む感じを補完する  

## 注意
- 本物のタイムマシンではありません
- 過去に戻ってやり直す機能はありません
- 未来を覗いて儲ける機能もありません
- 人生のやり直しボタンもありません

## 技術
- HTML
- CSS
- JavaScript（全部フロントエンド）
- 物理法則への挑戦（敗北済み）

## ライセンス
好きに使っていいです。  
ただし現実改変の責任は負いません。
```
